# 📝 To-Do List App (Python + Tkinter + SQLite)

A modern and user-friendly **To-Do List desktop application** built using Python.
This project features a clean GUI, persistent storage using SQLite, and smooth task management functionality.

---

## ✨ Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✔ Mark tasks as completed
* 🧹 Clear all tasks
* 💾 Persistent storage using SQLite
* 🎨 Modern dark-themed GUI
* ⌨️ Keyboard support (Press Enter to add task)
* 📊 Task counter (status display)

---



## 🛠️ Tech Stack

* **Python**
* **Tkinter** (GUI)
* **SQLite** (Database)

---

## 📂 Project Structure

```
todo-app/
│── main.py
│── tasks.db        # auto-created (ignored in git)
│── .gitignore
│── README.md
```

---

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/todo-app.git
```

2. Navigate into the project folder:

```
cd todo-app
```

3. Run the application:

```
python main.py
```

---

## ⚠️ Notes

* `tasks.db` is automatically created when you run the app
* Make sure it is included in `.gitignore`
* No external libraries required (uses built-in Python modules)
